Look Cleats

之前被鞋底板的問題困擾，因此在網路上問了一下
同時也做了一點survey…

Look的鞋底板(Cleats) 現在有分兩種。
分為Keo和Keo以前的，兩者不通用。

Keo的Cleats現在也有兩種款式，
一種是傳統平滑型，
另一種則有加上防滑片。
兩種通用。
有分紅色9度、灰色4.5度以及黑色0度三種。

而Keo以前的Cleats則叫做Look Delta。
分為紅色9度以及黑色0度兩種。


有Free ARC系統的Look卡踏(如我使用的pp396)
需要搭配紅色9度的Cleats
才能使用Free ARC的旋鈕調整0度、3度、6度以及9度四段可旋轉角度(angular float)。
若有Free ARC的踏板搭配黑色0度Cleats，
則Free ARC會失效，不管怎麼調整都是0度。
 
 
現在的Look keo系列似乎已經取消Look ARC的設計
改用鞋底板來調整。
我推測應該是為了減輕踏板重量的關係…。


拿之前買的Exustar Cleats出來比一比…



可以明顯看出有許多的不同，
下方曲線設計差異很大，
更別說長度整整短了一節。
難怪騎一騎會滑脫出來。

這個故事告訴我們：
鞋底板還是買原廠的好啊…。XD Read on →

中科水塔

今天太興奮了，新輪組，新鞋子。
擋不住誘惑，晚上蹺課去騎車啦>O<

今天騎的路線是傳說中的休閒級路線：中科水塔。
看著googlemap規劃好路線後就出發了。
預定走
文心路 >> 市政路 >> 環中路 >> 虹陽橋 >> 安和路 >> 福科路 >> 東大路 >> 科園路

新的卡鞋真的很棒，完全不會有壓迫的感覺。
而且不知道是不是錯覺，感覺穿新卡鞋
很自然就會有提拉的動作，
以前都要稍為意識，動作才會出來。
不過也有可能是剛好習慣了卡鞋吧。

騎到南屯路時，腳底板爆了，
轉進together買了浩悍的LOOK相容鞋底板
(以前用過wellgo和VP的)
在那邊借了工具當場安裝。

出來以後，發現入卡的感覺和以往不太一樣
而且很容易滑出來。
以前是很緊密的嵌合在一起，不會有空隙。
現在的感覺是卡住了，但前後左右有空隙可以移動，
很不習慣，回去再調整看看。

一路無話，順利到了福科路準備開始爬坡。
這時做了一件蠢事。
我以為我已經掛到小盤了，結果沒有。
覺得今天爬坡怎麼這麼累，明明看起來不陡啊。
右手趕快變檔…咦，怎麼已經用到27T的救命檔了。
爬上東大路時才發現原來我一直掛著大盤在爬…囧。
無緣無故的累積了很多的乳酸…。

接著要上科園路了，一路被登山車追過。
被超過時我就假裝很認真在觀察旁邊的建築物…(汗)
用自己的步調慢慢龜上去，心跳和呼吸都還可以。
不過膝蓋有一種很熱很熱的感覺，以前沒有過。

上到頂看到一堆人在看夜景聊天，
有騎單車的、有開車的、也有騎摩托車的。
這邊夜景真的很漂亮，路燈也很亮。
又沒有煞風景的鄉村卡啦OK…
難怪望高寮現在看夜景的人變少很多。

回程原本想走望高寮，但路不熟，
又忘記帶GPS，就走西屯路去136喝個甜的，
翻翻雜誌。

重編後的輪組很棒，順暢度一百分。
雖然讓我等了很久，但現在覺得都是值得的。
棘輪聲音好大，聽起來好爽。很有攻擊性的感覺。
