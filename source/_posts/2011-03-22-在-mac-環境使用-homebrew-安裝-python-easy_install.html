今天傳說中的 python 達人 / vim之神 Tiberius 大大 在公司主講了 python 基本教學，讓大家眼界都開了不少。

回家想在 mac 上安裝 python 卻發現有些問題。

使用 homebrew 安裝 python (2.7) 的話，會讓原本的 easy_install (2.6) 失效，因此要稍微做一點處理：

brew install python
之後若執行 easy_install ，會出現如下的錯誤： 
python version 2.6.1 can't run ./easy_install.  Try the alternative(s):

Run "man python" for more information about multiple version support in
Mac OS X. 這時只要去 setup-tools 的網站下載版本符合的安裝檔之後執行安裝： 

sh setuptools-0.6c11-py2.7.egg
就會把 easy_install 安裝在 homebrew 的 Celler 裡面了。
還沒結束呢！要在 .profile 裡面把 Celler 的 python bin 路徑加上去： 
export PATH=`brew --prefix python`/bin:$PATH
這樣就可以正常使用了！
